# 七大罪：魔神防線 (Seven Deadly Sins: Demon Defense Line)

簡短介紹：這是一款用 Python 製作的混合型遊戲，結合塔防 (Tower Defense) 與自走棋 (Auto Battler) 機制。玩家招募角色並組成羈絆，於每波敵人來襲時保衛領地。

（請在此處放置遊戲截圖或 GIF）

**遊戲特色**
- **自走棋機制**：每回合獲得金幣，在商店購買棋子，3 個相同星級的棋子會自動合成升級（最高 3 星）。
- **豐富羈絆系統**：包含種族（魔神族、女神族、巨人族、妖精族、人類）與陣營（七大罪、十戒、聖騎士、德魯伊）等多種組合加成。
- **波次防守**：敵人沿路徑進攻，需策略性放置近戰與遠程單位。
- **經濟策略**：利息與購買經驗可提升資金與人口上限。
- **完整音效與設定**：含背景音樂、介面音效與音量設定。

**需求**
- **Python**：3.x
- **Pygame**：用於畫面與音效處理

**安裝與執行**
1. 安裝 Pygame：

```powershell
pip install pygame
```

2. 啟動遊戲（在專案根目錄執行）：

```powershell
python game.py
```

**操作說明**
- **左鍵（拖曳）**：從商店購買棋子並拖入戰場或備戰區；可調整場上棋子位置。
- **右鍵**：點擊棋子以出售並回收金幣。
- **ESC**：遊戲中開啟暫停選單；暫停菜單中可返回或繼續遊戲。

介面功能：
- **刷新商店**（$2）：更換商店棋子。
- **購買經驗**（$4）：提升玩家等級，增加人口上限並提高高費卡出現機率。
- **出售區**：將棋子拖到指定區域以出售。

**部分羈絆範例**
- **魔神族**：2 / 4 / 6 — 顯著提升攻擊力。
- **女神族**：2 / 4 — 每秒回復生命值。
- **人類**：2 / 4 / 6 — 提升暴擊率。
- **巨人族**：2 / 3 — 顯著提升生命值。
- **七大罪**：3 / 5 / 7 — 全屬性提升（攻擊、生命、冷卻縮減）。
