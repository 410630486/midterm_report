# midterm_report
🛡️ 七大罪：魔神防線 (Seven Deadly Sins: Demon Defense Line)這是一款結合 塔防 (Tower Defense) 與 自走棋 (Auto Battler) 機制的 Python 遊戲。玩家需要招募《七大罪》中的角色，湊齊種族與陣營羈絆，抵禦魔神族的入侵！(您可以在此處替換成遊戲的實際截圖)✨ 遊戲特色自走棋玩法：每回合獲得金幣，在商店購買棋子。湊齊 3 個相同星級的棋子可自動升級（最高 3 星）。豐富羈絆系統：種族：魔神族、女神族、巨人族、妖精族、人類。陣營：七大罪、十戒、聖騎士、德魯伊。不同的羈絆組合會提供強力的屬性加成（如：攻擊力、攻速、回血、範圍等）。波次防守：敵人會沿著路徑進攻，玩家需策略性地將近戰單位部署在路徑上阻擋，遠程單位部署在高台輸出。經濟運營：利用利息機制存錢，或花費金幣購買經驗值來提升人口上限。完整音效：包含背景音樂、打擊音效、介面互動音效。設定系統：可隨時暫停調整音量與畫面亮度。🛠️ 安裝與執行1. 環境需求Python 3.xPygame2. 安裝依賴套件請開啟終端機 (Terminal) 或命令提示字元 (CMD)，輸入以下指令安裝 Pygame：Bashpip install pygame
3. 檔案準備請確保您的專案目錄結構如下，且 audio 資料夾內包含正確的音效檔：PlaintextGame/
├── game.py          <-- 遊戲主程式 (請將您的程式碼命名為此，或對應名稱)
├── README.md        <-- 本說明文件
└── audio/           <-- 音效資料夾 (必須存在)
    ├── bgm.mp3              # 背景音樂
    ├── click.wav            # 點擊音效
    ├── buy.wav              # 購買/升級音效
    ├── sell.wav             # 出售音效
    ├── refresh.wav          # 刷新商店音效
    ├── attack_melee.wav     # 近戰攻擊音效
    ├── attack_range.wav     # 遠程攻擊音效
    ├── dead.wav             # 敵人死亡音效
    └── game_over.wav        # 遊戲結束音效
注意： .wav 檔案必須是標準 PCM 編碼格式，否則 Pygame 可能無法讀取。4. 啟動遊戲在終端機中執行以下指令：Bashpython game.py
🎮 操作說明基本操作滑鼠左鍵 (拖曳)：從商店購買棋子拖入戰場或備戰區。調整場上棋子的位置。近戰單位 只能放在 道路上。遠程單位 只能放在 草地上。滑鼠右鍵：點擊棋子可直接 出售 並回收金幣。ESC 鍵：遊戲中 -> 暫停選單。暫停/設定中 -> 返回 / 繼續遊戲。介面功能刷新商店 ($2)：更換一批新的棋子。購買經驗 ($4)：提升玩家等級，增加人口上限（場上可放置的單位數量）並提高高費卡機率。出售區：將棋子拖曳至右下角紅色區域即可出售。📊 羈絆列表 (部分)羈絆名稱觸發條件效果魔神族2 / 4 / 6大幅提升攻擊力。女神族2 / 4每秒回復最大生命值。人類2 / 4 / 6提升暴擊率。巨人族2 / 3大幅提升生命值。七大罪3 / 5 / 7全屬性提升 (攻擊、生命、冷卻縮減)。低級魔族2部署時返還 1 金幣 (經濟型羈絆)。(更多羈絆請在遊戲中探索！)📝 開發日誌Step 69: 實裝 ESC 鍵全域控制（暫停/返回）。Step 67-68: 修復 UI 顯示層級與初始化崩潰問題，恢復裝飾物顯示。Step 63-66: 實裝 SoundManager 音效系統與防呆機制。Step 50-57: 完成全螢幕地圖繪製與 UI 佈局優化。
